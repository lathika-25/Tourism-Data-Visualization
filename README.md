# Tourism Data Visualization using Choropleth Map

## Project Overview
This project focuses on analyzing country-wise tourism data from 2014 to 2020. The data is processed and transformed to calculate the percentage growth in tourist arrivals. An interactive choropleth map is created to visualize tourism trends across different countries.

---

## Objective
To develop a choropleth map that visualizes the percentage growth in tourism across countries, enabling better understanding and comparison of global tourism trends.

---

## Tools & Technologies
- Python  
- Pandas  
- Plotly  

---

## Dataset
The dataset contains country-wise tourist arrival data from 2014 to 2020.

Columns used:
- Country  
- 2014, 2015, 2016, 2017, 2018, 2019, 2020  

---

## Methodology
1. Data Loading using Pandas  
2. Data Transformation (Wide to Long format)  
3. Data Cleaning (removing invalid entries)  
4. Percentage Growth Calculation  
5. Visualization using Choropleth Map  

---

## Output
- Interactive choropleth map showing percentage growth in tourism  
- Year-wise visualization using animation  
- Hover feature to display detailed data  

---

## Project Files
- tourism_case_study.csv (Dataset)  
- main.py / notebook.ipynb (Code)  
- Tourism_Choropleth_Map.html (Output visualization)  
- Case_Study_Report.pdf (Final report)  

---

## Key Insights
- Tourism growth varies across countries  
- Percentage growth provides better comparison than raw values  
- Significant decline observed in 2020  
- Visualization improves understanding of global trends  

---

## How to Run
1. Install required libraries:
   pip install pandas plotly

   python main.py

   Tourism_Choropleth_Map.html
